**[Voyageur](/) is a POC from 1 french student.**<br/>
Voyageur is a travel journal application that allows you to track and document your trip, add photos and notes for each place you visit. The application also offers travel recommendations and allows you to view / reviews your d photos from other users. Finally, it offers you a simple way to create a story of your trip in one place, with photos, videos and notes.

The app is using 2 API's, the first one is design to get all data concerning the countries and their culture / customs. The seconde one is design to get all data concerning promos on last minutes trips reservations.

## Installation

We use [React Native](https://reactnative.dev/) as an open-source platform for making universal native apps for Android, iOS, and the web with JavaScript and React.

- [ReactNative](https://reactnative.dev/) version 0.70.X or later
- [React](https://reactjs.org) version 18 or later
- [NodeJs](https://nodejs.org) version 18 or later

### Dependencies

Install the dependencies with yarn or npm

```bash
npm install or yarn insrtall
```

### Database setup

We use a [MySQL](https://www.mysql.com) database for the project:

- [phpMyAdmin alwaysdata](https://phpmyadmin.alwaysdata.com/) on Cloud (recommended)
- [MySQL](https://www.mysql.com) on our computer with the [SQL dump file](/db_voyageur.sql)
